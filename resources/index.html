examples

  • Hello World
  • Counting Servlet (demonstrates REQUEST, SESSION & FLASH scopes)
  • Scoped Wrapper (demonstrates scoping Servlets as REQUEST or SESSION, using an @Singleton Wrapper)
  • Request Parameters (injection of request params)
  • Info Servlet (demonstrates compliance of servlets managed by warp-servlet)
  • Info Servlet (same as above, but with a *.info mapping)
  • Unmanaged Servlet (a "normal" servlet NOT managed by warp-servlet, to compare with above)
  • Unmanaged Servlet #2 (same as above, but with a *.uninfo mapping)
  • Request Dispatching Servlet (a managed servlet that includes a JSP)
  • Request Dispatching in reverse (a JSP that includes a managed servlet)
  • Request Forwarding (a managed servlet that forwards to a JSP--sampleInclude.jsp)
  • Request Forwarding (a JSP that forwards to a managed servlet)
